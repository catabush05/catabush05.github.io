about!

I'm a Junior at Willamette University majoring in Data Science. I'm originally from Tacoma, Washington, but I've also lived in Mijas, Spain and Syracuse, NY. I'm part Colombian and fluent in both English and Spanish. Soccer has always been an integral part of my family and some of my favorite memories revolve around the sport: watching Real Madrid at the Bernabeu, United at Old Trafford, and the Concacaf Champions League final at Lumen Field.

Some time ago I was drawn into soccer analytics and I never looked back. I've used everything from Python, R, and SQL to Tableau. I'm skilled in crafting beautiful data visualizations and building automated processes from the ground up. Keep scrolling to see the cool things I've built!

ASA VizHub

A platform for generating industry-quality data visualizations at the click of a button. Built using ASA data and models.

R Shiny ggplot2 duckDB JavaScript SQL

Key Features

  • 20+ custom plots in ggplot
  • Thousands of games across 6 leagues
  • Daily data updates
  • Downloadable images
  • Beautiful, interactive interface

@mlsstat

An automated Bluesky feed posting data-based post-match reports for MLS matches.

Python Matplotlib SQL

Key Features

  • Robust data pipeline
  • Automated posting to Bluesky
  • Free, public, and accessible
  • High-quality visuals

@nwslstat

An automated Bluesky feed posting data-based post-match reports for NWSL matches.

Python Matplotlib SQL

Key Features

  • Women's soccer coverage
  • Entirely original charts/plots
  • ASA g+ integration
  • Automatic monitoring in Discord

@uslstat

An automated Bluesky feed posting data-based post-match reports for USL C, USL 1, and USL S matches.

Python Matplotlib SQL

Key Features

  • Lower league coverage
